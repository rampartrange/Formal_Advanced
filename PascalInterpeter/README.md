# Интерпретатор языка Pascal (Андрианов Артем Б05-924)
### Статус: **в доработке**

### Сделано на данный момент:
1. Лексический анализатор **_scanner.l_**
2. Парсер грамматики **_parser.y_**
3. Создан объект **_PascalObject_**, благодаря которому поддерживаются:
  - Объекты типов: **Integer**, **String**, **Boolean** и **Double**
  - Операции присваивания
  - Арифметические операторы (в том числе конкатенация строк и унарный минус)
  - Операции __div__ и __mod__
  - Булевы операции __and__, __or__, __xor__, __not__
  - Операции сравнения 
  - Явный каст каждого из типа к bool
  - Поддержка базового синтаксиса программы на языке Pascal.
 
### В процессе доработки:
1. Написание классов для поддержки и обработки Control Flow Statements.   
    В данный момент столкнулся с проблемой поддержки вложенных циклов и if-ов, так же до конца еще не продумана архитектура класса и его взаимодействие с внешними модулями.
2. Поддержка зарезервированных имен в языке Pascal. Построение символьной таблицы.
3. Рефакторинг грамматики для поддержки синтаксиса программ в Pascal.
4. Обработка ошибок при несоответсвии типов.

### Запуск программы:
```
mkdir build
cd build 
cmake .. && make
./PascalInterpreter <path/to/file>
```
На данный момент граматикой поддерживаются только входные данные вида:
```Pascal
var
a, b : real;
c : boolean; // Обьявление типов.
d : integer;

begin
b := 3 > 2;
a := false and 1;
c := true and (0); // Набор assigment'ов. Корректное вычисление
d := 4 mod 2;
str := 'apple';
end.
```

Вывод программы будет иметь вид:
```
b: REAL
a: REAL  
c: BOOLEAN
d: INTEGER // Результат парсинга и обработки части кода с Declarations
BOOLEAN: a := FALSE
BOOLEAN: b := TRUE
BOOLEAN: c := FALSE
INTEGER: d := 0
STRING: str := apple // Результат вычисление всех assigment'ов в MainBlock

```
